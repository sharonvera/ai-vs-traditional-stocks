# AI vs Traditional Stocks: Quantitative Analysis (2020–2026)

## Overview
This project analyzes AI-related equities vs traditional defensive stocks using Python-based financial modeling.

---

## Data
- AI Portfolio: NVDA, MSFT, AMD, META, GOOGL, AVGO  
- Traditional Portfolio: KO, PG, WMT, JPM, XOM, JNJ  
- Benchmark: SPY  

---

## Methods
- Portfolio construction  
- Cumulative returns analysis  
- Volatility comparison  
- CAPM-style regression  
- Market benchmark comparison  

---

## Key Results
- AI portfolio significantly outperforms traditional stocks  
- Higher volatility and higher market sensitivity (beta > 1)  
- Strong relative performance vs SPY  
- Risk-return tradeoff favors AI over sample period

Includes:
- Cumulative returns
- AI vs SPY comparison
- Volatility comparison

## Tools Used
Python, pandas, numpy, matplotlib, statsmodels, yfinance
